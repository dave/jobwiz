## Data Sources

This template requires data from:
- Company module for {Company} (#11)
- Role module for {Role} (#13)
- Glassdoor interview reviews filtered by {Company} + {Role}
- levels.fyi for compensation data
- Blind for anonymous employee insights
- LinkedIn for org structure research

## Usage Notes

- Only generate combined modules for high-traffic company+role combinations
- Use search volume data (#5) to prioritize which combinations to generate
- Cross-reference company and role modules to avoid duplication
- This is premium content (isPremium: true)
- Update more frequently than base modules (quarterly)
- Flag for review if source data is sparse
