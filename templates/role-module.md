# Role Module Template

## Metadata

```yaml
name: "{role} Interview Prep"
moduleType: role
description: Role-specific interview preparation for {role} positions
roleCategory: "{role-category}"
```

---

## Sections

### 1. Role Overview

- **Required:** true
- **Block Types:** header, text, infographic
- **Estimated Word Count:** 300

What this role does and why companies hire for it.

**Example Content:**

```
# The {Role} Role

A {Role} is responsible for {core responsibilities}. Companies hire {Role}s to {business value they provide}.

**Core Functions:**
- {Function 1}
- {Function 2}
- {Function 3}

[Infographic: {Role} Career Path and Levels]

**Market Insight:** {Role} positions are currently {market demand context - growing, competitive, etc.}
```

## Role Categories

This template applies to these role types:
- Software Engineer (SWE)
- Product Manager (PM)
- Data Scientist
- Designer (UX/UI/Product)
- Marketing
- Sales
- Finance
- Operations
- Human Resources

## Usage Notes

- Technical sections vary significantly by role type
- Assessment sections should reference actual formats used (coding, case studies, design challenges, etc.)
- Salary data should be sourced from levels.fyi, Glassdoor, Blind
- Update salary ranges annually
- Cross-reference with company modules for company-specific variations
